# Keybindings

This document outlines the default keybindings for the `sahs` Hyprland configuration.

## Modifier Key
The main modifier key is `SUPER` (the "Windows" key).

## Application Bindings
| Action | Binding |
| :--- | :--- |
| Open Terminal | `SUPER + RETURN` |
| Open File Manager | `SUPER + SHIFT + F` |
| Open Browser | `SUPER + SHIFT + L` |
| Open Private Browser | `SUPER + ALT + SHIFT + L` |
| Open App Menu (Wofi) | `SUPER + SPACE` |
| Kill Active Window | `SUPER + W` |
| Exit Hyprland | `SUPER + ESCAPE` |

## Brightness Controls (Submap)
To use brightness controls, first press the **Trigger**, then the **Sub-key**. The submap will reset after any other key press.

### 10% Step Adjustments
**Trigger:** `SUPER + CTRL + F1`

| Action | Sub-key |
| :--- | :--- |
| Increase Brightness | `Up Arrow` |
| Decrease Brightness | `Down Arrow` |
| Get Current Level | `C` |
| Set Max Brightness | `M` |

### 5% Step Adjustments
**Trigger:** `SUPER + CTRL + ALT + F1`

| Action | Sub-key |
| :--- | :--- |
| Increase Brightness | `Up Arrow` |
| Decrease Brightness | `Down Arrow` |

## Window Management
| Action | Binding |
| :--- | :--- |
| Move Focus (Arrows) | `SUPER + [Left/Right/Up/Down]` |
| Move Window to Workspace | `SUPER + SHIFT + [0-9]` |
| Toggle Floating | `SUPER + V` |
| Toggle Split | `SUPER + J` |

## Workspaces
| Action | Binding |
| :--- | :--- |
| Switch Workspace | `SUPER + [0-9]` |
| Toggle Special Workspace | `SUPER + S` |
| Move to Special Workspace | `SUPER + SHIFT + S` |
